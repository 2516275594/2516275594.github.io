技术说明

由于浏览器的同源策略限制，无法直接访问跨域API。这里使用了CORS代理服务来解决此问题。

如果您需要更稳定的解决方案，可以：

  1. 部署自己的代理服务器（Node.js示例代码已提供）
  2. 使用支持CORS的API服务
  3. 在服务器端直接调用API
查看Node.js代理服务器代码 
const express = require('express');
const cors = require('cors');
const axios = require('axios');

const app = express();
app.use(cors());

app.get('/proxy/api/img/tu/girl', async (req, res) => {
    try {
        const response = await axios.get('https://api.52vmy.cn/api/img/tu/girl');
        res.json(response.data);
    } catch (error) {
        res.status(500).json({ error: error.message });
    }
});

const PORT = 3000;
app.listen(PORT, () => {
    console.log(`代理服务器运行在 http://localhost:${PORT}`);
});
